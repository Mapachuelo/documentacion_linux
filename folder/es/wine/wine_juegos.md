Crear una lista de parametros de lanzamiento para juegos en steam.

Requisitos:
- Linux.
- goverlay.
- mangohud.
- vkbasalt.
- gamescope.

La instalación de estos son:
## Arch linux 
Se requiere el repositorio de cachyos para vkbasalt.
```
sudo pacman -S mangohud vkbasalt-cli gamescope goverlay 
```
O por mediante AUR.
```
yay -S vkbasalt
```
