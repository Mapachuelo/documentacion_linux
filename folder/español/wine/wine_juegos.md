Crear una lista de parametros de lanzamiento para juegos en steam.

Requisitos:
- Linux.
- goverlay.
- mangohud.
- vkbasalt.
- gamescope.

La instalación de estos son:
## Arch linux 
```
sudo pacman -S mangohud vkbasalt-cli gamescope goverlay 
```
Se requiere el repositorio de cachyos para vkbasalt, o por mediante AUR es 
```
yay -S vkbasalt-cli
```
