### Ver ramas de git
git branch

### Cambiar a otra rama
git switch "nombre de la rama"

### Borrar rama existente
git branch -D "nombre de la rama"

### Para crear una rama nueva y saltar a ella de inmediato:  
git switch -c nueva-rama

### Limpiar ramas que ya no existen
git fetch --all --prune

### Actualizar ramas
git fetch
